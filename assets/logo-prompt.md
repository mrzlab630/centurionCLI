# CENTURION Logo Generation Prompts

## Primary Prompt (Recommended)

```

cyberpunk aesthetic. The flag displays "CENTURION" text with glowing
neon cyan circuit patterns. Purple and gold color scheme with scan lines.
Dark background with subtle grid. Atari 2600 era graphics style.
Simple, iconic, retro-futuristic. 128x128 pixels.
```

## Recommended Tools

- Midjourney: Add `--style raw --v 6` for cleaner pixel art
- DALL-E 3: Works well with detailed pixel art descriptions
- Stable Diffusion: Use pixel art LoRA models
- Leonardo.ai: Good for game asset style

## Usage Notes

1. Generate at higher resolution, then downscale for crisp pixels
2. Add `transparent background` if needed for GitHub
3. Request multiple variations to choose from
4. For favicon: generate 32x32 or 64x64 version
