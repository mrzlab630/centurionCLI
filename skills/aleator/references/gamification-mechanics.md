# ALEATOR — Gamification Mechanics Catalog

> 50 engagement mechanics organized by category with psychology, implementation notes, and ethical rating.

## Progression Mechanics

| Mechanic | Psychology | How It Works | Ethical |
|---|---|---|---|
| **Points** | Accomplishment, feedback | Numeric score for actions | ✅ |
| **Levels** | Competence, status | Tiered progression (XP thresholds) | ✅ |
| **Progress bars** | Goal gradient, completion drive | Visual % to next milestone | ✅ |
| **Skill trees** | Autonomy, mastery | Branching upgrade paths | ✅ |
| **Milestones** | Achievement, dopamine spikes | Significant progress markers | ✅ |
| **Prestige/Reset** | Fresh challenge, status | Voluntary restart at higher tier | ✅ |

**Goal Gradient Effect:** People accelerate effort as they approach a goal. A progress bar at 80% drives more action than one at 20%.

## Social Mechanics

| Mechanic | Psychology | How It Works | Ethical |
|---|---|---|---|
| **Leaderboards** | Competition, social comparison | Ranked user lists | ⚠️ can demotivate low-rank |
| **Teams/Guilds** | Relatedness, obligation | Group membership with shared goals | ✅ |
| **Friend challenges** | Social proof, competition | Head-to-head or cooperative tasks | ✅ |
| **Gifting** | Reciprocity, relatedness | Send rewards to other users | ✅ |
| **Activity feed** | Social proof, FOMO | "User X just earned Y" | ⚠️ can create envy |
| **Mentoring** | Authority, relatedness | Experienced users help newbies | ✅ |
| **Referrals** | Reciprocity, social proof | Invite friends → both get rewards | ✅ |
| **Shared milestones** | Unity, celebration | Community goals ("We hit 1M trades!") | ✅ |

**Leaderboard best practices:**
- Show relative position (nearby ranks), not just top 10
- Multiple leaderboards (weekly resets, categories) so everyone can compete
- Opt-in, not forced

## Reward Mechanics

| Mechanic | Psychology | How It Works | Ethical |
|---|---|---|---|
| **Daily login bonus** | FI schedule, routine | Reward for opening app each day | ✅ |
| **Streak rewards** | Loss aversion, commitment | Consecutive-day bonuses (escalating) | ⚠️ needs streak shields |
| **Loot boxes/Mystery rewards** | VR schedule, curiosity | Random contents on open | ⚠️ if paid → gambling |
| **Spin wheels** | VR schedule, visual excitement | Random reward from visual spinner | ⚠️ needs odds disclosure |
| **Pity timers** | Frustration prevention | Guaranteed reward after N misses | ✅ essential for VR |
| **Bonus multipliers** | Urgency, excitement | 2x rewards for limited time | ⚠️ if creates pressure |
| **Surprise drops** | Delight, return rate | Random unexpected rewards | ✅ |
| **Tiered rewards** | Goal gradient, aspiration | Better rewards at higher tiers | ✅ |

## Temporal Mechanics

| Mechanic | Psychology | How It Works | Ethical |
|---|---|---|---|
| **Daily quests** | Routine, FI schedule | New tasks every 24h | ✅ |
| **Weekly challenges** | Longer commitment | Bigger tasks, bigger rewards (7-day cycle) | ✅ |
| **Seasonal events** | Scarcity, novelty | Limited-time content/themes | ✅ |
| **Countdown timers** | Urgency, loss aversion | Visible time remaining | ⚠️ must be real |
| **Cooldown periods** | Anticipation, patience | "Come back in 4h for next reward" | ✅ if reasonable |
| **Energy systems** | Return rate, monetization | Limited actions per time period | ⚠️ if pay-to-refill |

## Mechanic Combinations (Recipes)

### "The Streaker" (Retention)
```
Daily Login + Streak Counter + Escalating Rewards + Streak Shield (loss aversion)
Schedule: FI (daily) + loss aversion
Best for: Daily active users metric
```

### "The Competitor" (Engagement)
```
Leaderboard + Weekly Reset + Tiered Rewards + Social Sharing
Schedule: FI (weekly) + social proof
Best for: Power users, competitive audiences
```

### "The Collector" (Investment)
```
Badges + Collections + Rare Items + Trading + Profile Display
Schedule: VR (rare drops) + FR (milestone badges)
Best for: Long-term retention, identity building
```

### "The Explorer" (Discovery)
```
Easter Eggs + Unlockables + Branching Paths + Mystery Rewards
Schedule: VR (random discovery)
Best for: Curiosity-driven audiences, content-rich products
```

### "The Socializer" (Virality)
```
Referrals + Team Challenges + Gifting + Activity Feed + Shared Milestones
Schedule: Social triggers
Best for: Network effects, organic growth
```

### "The Achiever" (Mastery)
```
Skill Tree + Quests + Certificates + Stats Dashboard + Mastery Levels
Schedule: FR (quest completion) + competence feedback
Best for: Skill-based products, education, professional tools
```
