# ORATOR — TikTok Platform Reference

> Source: Sprout Social 2026, Buffer, DarkRoom Agency, OpusClip, Socialync

## Algorithm Mechanics (2026)

### Distribution Flow
1. New video shown to **existing followers first** (changed from 2024)
2. Algorithm measures: completion rate, shares, saves among followers
3. If performance good → pushed to non-followers via FYP
4. Each expansion round tests larger audience pools

### Key Ranking Signals (by priority)
| Signal | Weight | Notes |
|---|---|---|
| Completion rate | HIGHEST | Need 70%+ for viral potential (up from 50% in 2024) |
| Rewatch/loop | Very High | "I watched this 3 times" = gold |
| Shares | Very High | Indicates intent to distribute |
| Saves | High | Indicates intent to return |
| Comments (meaningful) | High | Substantive > emoji comments |
| Likes | Medium | Baseline signal |
| "Not interested" | NEGATIVE | Strong negative signal |

### Content Signals
- Algorithm "listens" to video audio for topic classification
- Keyword-rich captions = search optimization (TikTok is a search engine now)
- Niche specificity = better audience matching
- Trending sounds get algorithmic boost

## Content Formats

### Video Script Template
```
[0-2s] HOOK — Stop the scroll. Question, shock, or promise.
[2-15s] CONTEXT — Set up the problem or topic.
[15-60s] VALUE — Deliver the insight, tutorial, or story.
[Last 3s] CTA — "Follow for more" / "Comment your X" / loop-back to hook.
```

### Optimal Length (2026)
- **60-180 seconds** outperforming 15-second clips
- Short (15s) still works for trends/memes
- Long (3min+) only for highly niche expert content

### Content Types That Work
1. **Tutorials/How-to** — "Here's how to [specific thing]"
2. **Storytime** — Personal experiences, narrative arc
3. **Duets/Stitches** — React to trending content
4. **Before/After** — Transformation content
5. **POV** — "POV: you're a [relatable scenario]"
6. **Green screen** — Screenshots/charts with commentary (great for crypto)

## Crypto/Trading Bot Specifics

### What Works
1. **Trading win screenshots** — with green screen effect, reaction
2. **Tutorial walkthroughs** — "How to copy-trade in 60 seconds"
3. **Market analysis** — charts + commentary (green screen)
4. **Memes** — Solana culture, degen humor
5. **Behind the scenes** — bot development, team culture

### What Fails
1. Overly technical content (audience is broader/younger)
2. Financial advice or promises
3. Corporate/polished tone (TikTok rewards authenticity)
4. Static images without narration

## Timing & Frequency

- **Frequency:** 3-5 videos/week (consistency > volume)
- **Peak:** Varies by audience — check Analytics > Followers > Most Active
- **General peaks:** 7-9 AM, 12-3 PM, 7-11 PM (local audience timezone)
- **Consistency:** Algorithm punishes long gaps more than low frequency

## Engagement
- Reply to EVERY comment in first hour ("active creator" signal)
- Pin best comment (drives more discussion)
- Reply with video (algorithmic boost for reply videos)
- Engage with trending content in your niche daily

---

## Anti-Patterns
- Recycling Instagram Reels with watermarks
- Not hooking in first 2 seconds (80% of viewers decide here)
- Posting inconsistently (week on, week off = algorithm reset)
- Ignoring comments (signals inactive creator)
- Using copyrighted music (shadowban risk)
- Overediting/overproducing (TikTok rewards raw/authentic)
