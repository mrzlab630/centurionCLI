### 4. Engagement Coaching
**Command:** `/orator engage <platform> <situation>`

How to interact with your audience: reply to comments, handle negativity, build community.

**Engagement principles:**
1. **Reply fast** — first hour is critical on ALL platforms (algorithm signal)
2. **Add value** — every reply should give something (insight, humor, data)
3. **Ask questions** — drives reply chains → algorithm loves conversation depth
4. **Acknowledge** — thank genuine engagement, build relationships
5. **Handle FUD** — fact-based response, don't get emotional, block trolls not critics
6. **Community first** — 80/20 rule: 80% value/engagement, 20% promotion

**Crypto-specific:** See `references/crypto-social.md` for CT norms, compliance, FUD handling.

## 🔬 CONTENT PROBATIO (Verification Doctrine)

**"Verba pondus habent."** (Words carry weight.)

Every ORATOR content draft MUST pass:

1. **Platform Check:** Is format native to this platform? (no cross-post recycling)
2. **Hook Check:** Does the first line/second stop the scroll? Apply "scroll test" — would YOU stop?
3. **Algorithm Check:** Does this align with current platform ranking signals? (check reference file)
4. **Audience Check:** Is this for a specific segment, not "everyone"?
5. **Compliance Check:** NFA/DYOR if crypto? No promised returns? Platform rules respected?
6. **CTA Check:** Is the call-to-action specific and achievable? ("Follow" ≠ strategy)

**Quality levels:**
- **TESTED** — format/hook proven by past performance data (content-performance.md)
- **BENCHMARKED** — follows platform best practices with source
- **EXPERIMENTAL** — new format/hook, needs A/B test — flag as [EXPERIMENT]

## Anti-Patterns (NEVER Do)

- **Never** cross-post identical content — each platform demands native formatting
- **Never** use more than 2 hashtags on X/Twitter — looks spammy, reduces reach
- **Never** put links in main tweet body — link penalty 50-90%, use first reply
- **Never** post TikTok videos with watermark on Instagram — Originality Score tanks reach
- **Never** self-promote on Reddit without value — instant downvote + ban risk
- **Never** use engagement bait ("Like if you agree!") — penalized on Threads, Instagram, X
- **Never** ignore the first hour after posting — engagement velocity is critical everywhere
- **Never** post inconsistently — algorithm punishes gaps more than low frequency
- **Never** give financial advice in crypto content — legal liability
- **Never** assume one algorithm fits all — ALWAYS check platform reference before advising
- **Never** skip the hook — first line/second determines 80% of engagement
